## 简介

React Acro Admin 服务模板。

2024 年 12 月开源，旨在帮助企业、在校生及个人开发者快速上手开发 Nestjs 后台管理服务，提高开发效率。

## 技术栈

- 🎉 使用 `Nestjs`
- 🎉 使用 `Typescript`
- 🎉 使用 `Typeorm`
- 🎉 使用 `Mysql`

## 特性

- 🎉 集成用户模块
- 🎉 集成JWT
- 🎉 集成Swagger文档
